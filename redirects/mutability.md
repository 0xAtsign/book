% Mutability

There is a new edition of the book and this is an old link.

> Variables are immutable only by default; we can make them mutable by adding mut in front of the variable name.

```rust
let mut x = 5;
println!("The value of x is: {}", x);
x = 6;
println!("The value of x is: {}", x);
```
